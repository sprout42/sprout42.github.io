### Update 2019-10-04

Ok, so I'm not an HTML or Javascript person, and when the blog didn't work right 
before DEF CON I got frustrated and put the blog aside until "later".  Well it's 
now "later". I found out I forgot to add the `_config.yml` file to this git repo 
which explains a lot about why it wasn't working. Oddly enough the project 
I used as a basis for my theme had `_config.yml` in the `.gitignore` file? Not 
sure why you'd do that. Anyway, the blog is up and functional! Now I finally 
have a place to add all those small notes and things that I keep writing down in 
text files but haven't actually put anywhere useful.
